# Arquitetura do Sistema

## Camadas

Frontend
Backend API
Motor simbólico
Agentes de IA
Banco de dados
Infraestrutura

## Fluxo principal

Usuário → Frontend → API → Motor simbólico → Agentes IA → Resposta

## Tecnologias recomendadas

Frontend:

* React
* Next.js
* Tailwind

Backend:

* Python
* FastAPI

Banco de dados:

* PostgreSQL
* Neo4j (grafo simbólico)

Infraestrutura:

* Docker
* Cloud (AWS / GCP)
