# Motor de Oráculo Inteligente

O oráculo gera tiragens contextualizadas.

## fatores de decisão

perfil do usuário
contexto da pergunta
posição na Árvore da Vida
aleatoriedade controlada

## fórmula de score

score_carta =

0.4 perfil_usuario
0.3 contexto_pergunta
0.2 posição_arvore
0.1 aleatoriedade

## tipos de tiragem

3 cartas — insight
5 cartas — decisão
7 cartas — evolução
