# Product Requirements Document

## Produto

Código do Eu Profundo

## Visão

Construir uma plataforma de autoconhecimento baseada em sistemas simbólicos estruturados e inteligência artificial.

## Público

* interessados em autoconhecimento
* terapeutas
* coaches
* estudantes de simbolismo

## Funcionalidades principais

1. geração do Código do Eu Profundo
2. mapa arquetípico do usuário
3. oráculo inteligente
4. mentor arquetípico
5. análise evolutiva

## Diferenciais

* sistema simbólico estruturado
* interpretação baseada em IA
* integração entre múltiplos sistemas tradicionais

## Métricas de sucesso

* usuários ativos
* retenção mensal
* número de consultas ao oráculo
* conversão para plano premium
