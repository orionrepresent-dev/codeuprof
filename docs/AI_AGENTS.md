# Arquitetura de Agentes

O sistema utiliza múltiplos agentes especializados.

## numerology_agent

Calcula números simbólicos.

## tarot_agent

Gera tiragens e interpreta arcanos.

## cabala_agent

Mapeia arquétipos na Árvore da Vida.

## astrology_agent

Interpreta padrões planetários.

## psychology_agent

Analisa arquétipos psicológicos.

## synthesis_agent

Unifica as interpretações.

## fluxo

Input do usuário → agentes especializados → agente de síntese → resposta final.
